# Hostinger Webmail Login Clone

This project is a replica of the Hostinger Webmail login interface, created to enhance frontend development skills with Next.js, Tailwind CSS and React. The focus is on reusable components, responsiveness and good development practices.

##Objective

To provide a foundation for frontend developers to explore concepts such as:

- Using reusable components in React.
- Integrating images and icons in Next.js.
- Managing state with hooks like `useState`.
- Modern styling with Tailwind CSS.

## Technologies Used

- **Next.js**: Framework for React applications.
- **Tailwind CSS**: Detailed styling library.
- **Lucide-react**: Modern and configurable icons.
- **TypeScript**: Adds static hinting to JavaScript.

## Features

- **Login form**:
- Username and password input fields.
- Show/hide password with dynamic icons.
- Basic validation and form state management.

- **Responsive**:
- Layout adjusted for large and small screens.

- **CTA (Call to Action)**:
- Link to password recovery.
- Button to get email plans.

## How to run the project

1. **Clone the repository**:

```bash
git clone https://github.com/your-username/hostinger-clone.git
```

2. **Install as dependencies**:

```bash
npm install
```

3. **Run the development server**:

```bash
npm run dev
```

4. **Access without a browser**:

```
http://localhost:3000
```
